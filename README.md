# 🧮 Calculator App

🚀 A simple, interactive, and responsive web-based calculator built using HTML, CSS, and JavaScript.

---

## 👋 About The Project
This project is a beginner-friendly calculator application that performs basic arithmetic operations like addition, subtraction, multiplication, and division.

It is designed with a clean UI and helps understand frontend development concepts like DOM manipulation and event handling.

---

## ✨ Features
- ➕ Addition  
- ➖ Subtraction  
- ✖ Multiplication  
- ➗ Division  
- 🧼 Clear / Reset button  
- 🖱️ Button click interaction  
- 📱 Responsive design  
- ⚡ Real-time output display  

---

## 🛠️ Tech Stack
- HTML → Structure  
- CSS → Styling  
- JavaScript → Logic

## 🎮 How to Use
- Enter numbers using buttons  
- Choose operator (+, -, ×, ÷)  
- Press "=" to get result  
- Press "C" to clear
